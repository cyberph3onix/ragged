# RAG101

A Retrieval-Augmented Generation (RAG) system built using:

- Python
- ChromaDB
- Sentence Transformers
- Qwen
- Local Embeddings

## Features

- PDF ingestion
- Document chunking
- Vector embeddings
- Semantic search
- Context-aware answer generation
